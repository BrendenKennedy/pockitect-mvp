NIST-Inspired Baseline (Summary)

Access Control
- Enforce least privilege and role-based access.
- Use MFA for privileged access.
- Review access periodically.

Configuration Management
- Standardize images and hardened baselines.
- Track changes and use version control.

Identification and Authentication
- Use unique identities; avoid shared credentials.
- Rotate keys and secrets regularly.

Logging and Monitoring
- Centralize logs; retain for a defined period.
- Alert on anomalous access and privileged actions.

Data Protection
- Encrypt in transit (TLS) and at rest.
- Classify data and restrict access accordingly.

Incident Response
- Maintain runbooks; test procedures.
- Capture evidence and support post-incident review.
